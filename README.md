# ProofReading

## Usage

setup (only do once per session)
```
> source setup.sh
```

Do the proof reading
```
py proofRead.py ../*tex
```

To ignore a given suggestion, add the suggestion hash to the .prI

```
> prIgnore a74118c148
```
